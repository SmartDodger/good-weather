Used technologies:
HTML5;
SCSS;
Angular5;
RxJS;
Bootstrap4;
ngx-bootstrap;

Приложение для получения прогноза погоды на 5 дней.
Работа с OpenWeatherMap API. Получене JSON данных и их обработка с помощью Angular5. Собственный дизайн с
адаптрированной версткой. Deploy на Heroku.
